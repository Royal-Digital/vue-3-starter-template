# Vue 3 Starter Template

> That's a Vue 3 starter template which has: Vue Router, Vuex, Sass and Webpack

## Install Depenedncies

```sh
npm install
```

or if you use Yarn

```sh
yarn install
```

## Serve the App

```sh
npm run dev
```

or if you use Yarn

```sh
yarn dev
```

## Build the App

```sh
npm run build
```

or if you use Yarn

```sh
yarn build
```
